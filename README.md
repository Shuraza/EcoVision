## Descrição

O **EcoVision** é um sistema web desenvolvido como projeto acadêmico com o objetivo de incentivar **conscientização ambiental e práticas de reciclagem**.

A plataforma permite que usuários:

* registrem **coletas de resíduos recicláveis**
* realizem **denúncias ambientais**
* acompanhem **coletas e denúncias registradas**
* aprendam mais sobre **reciclagem e descarte correto de materiais**

O sistema busca facilitar a comunicação entre cidadãos e iniciativas de preservação ambiental.

</details>

<details>
<summary>Funcionalidades</summary>

## Funcionalidades do Sistema

* **Página Inicial (Landing Page)**
  Apresentação da plataforma e suas funcionalidades.

* **Registro de Coletas**
  Permite cadastrar solicitações de coleta de resíduos recicláveis.

* **Registro de Denúncias**
  Possibilita registrar problemas ambientais como descarte irregular de lixo.

* **Acompanhamento de Coletas**
  Tela onde o usuário pode visualizar o status das coletas solicitadas.

* **Acompanhamento de Denúncias**
  Página para visualizar denúncias registradas.

* **Página Educativa**
  Contém instruções sobre reciclagem e separação correta de resíduos.

</details>
